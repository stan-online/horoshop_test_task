# Internal Dashboard

## Завдання

1. Запустити систему.
2. Знайти всі проблеми.
3. Виправити їх.
4. Надати список знайдених проблем.
5. Описати команди перевірки працездатності.

## Запуск

```bash
docker compose up --build
```

## Очікуваний результат

Після виправлення всіх проблем:

- `http://localhost/` => HTML-сторінка
- `http://localhost/api/` => JSON
- `http://localhost/api/health` => `{"status":"OK"}`

## Що здати

1. посилання на репо з виправленнями або архів репозиторія.
2. Звіт: список знайдених проблем (симптом -> причина -> fix).
3. Команди перевірки працездатності.

Використання ChatGPT, Claude Code, Cursor, Gemini та інших AI-інструментів дозволено. На інтерв'ю буде 15-хвилинне технічне обговорення рішення.
